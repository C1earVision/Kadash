Hardware Company ChatBot System:
User Functional Requerments:
  The chatbot is able to answer questions related to item availability, item comparisons 
  and recommend items based on user prefrences.
Company Functional Requerments:
  1. The chatbot is able to answer questions related to sales statistics.
  2. The cahtbot is able to execute commands like schedualing meetings or ordering items.
--------------------------------------------------------------------------------------------
Folder Structure:
main.py
  server to listen for user input and pass it to application workflow
.env
  contains API keys
Agent
  AgentWorkFlow.py
    returns workflow of the application
Tools
  dataBaseQuery.py
    Turns the user input into data base query and contacts backend server for query results 
    and returns the resluts.
  webSearch.py
    Makes a websearch and returns the results.
  schedualMeeting.py
    contacts the neccessary department members and notifies them of the meeting time.
  weeklyReport.py
    Pulls metrics (e.g., sales, tickets, KPIs) and makes a report with suggestions
    on how to move forward.
  OrderSupplies:
    Detects when office supplies or stock run low (via thresholds).
    Automatically reorders or gets approval.
Utils:
  contains additional tools neccessary for the execution of primary tools.
